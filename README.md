# CHARACTER_CREATOR

A simple Python program that creates a game character based on user input.

## Features
- Takes character name and stats (strength, intelligence, charisma)
- Validates input values
- Ensures total stats equal 7
- Displays stats using visual bars (● and ○)

## Rules
- Name must be a string, max 10 characters, no spaces
- Stats must be integers between 1 and 4
- Total of all stats must equal 7

## How to Run
bash
python character_creator.py

Alex
